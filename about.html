اسم المشروع: القاموس السيبراني التفاعلي

فكرة المشروع: يهدف المشروع إلى إنشاء قاموس تفاعلي يشرح مصطلحات الأمن السيبراني باللغتين العربية والإنجليزية، مع توفير تعريفات مبسطة، أمثلة واقعية، حلول مقترحة، ومساعد ذكي يساعد المستخدم على فهم المصطلحات بشكل أسرع.

أهداف المشروع

  • تسهيل فهم مصطلحات الأمن السيبراني للمستخدم العربي.
  • تقديم تعريفات مبسطة مناسبة للمبتدئين.
  • إضافة أمثلة واقعية وسياق سعودي للمصطلحات.
  • دعم المستخدم من خلال مساعد ذكي مرتبط بـ n8n.
  • توفير تجربة استخدام واضحة ومناسبة لكبار السن.

الأدوات والتقنيات المستخدمة

  • HTML لبناء صفحات الموقع.
  • CSS لتصميم الواجهة وتحسين تجربة المستخدم.
  • JavaScript لتشغيل البحث والفلاتر والتفاعل.
  • Google Sheets كقاعدة بيانات للمصطلحات.
  • n8n لربط Smart AI Assistant بالموقع.

الفئة المستهدفة

الطلاب، المبتدئون في الأمن السيبراني، المستخدمون العرب، وكبار السن الذين يحتاجون إلى واجهة واضحة وسهلة القراءة.
